Forward proxy mode

In this mode, mitmproxy accepts proxy requests and unconditionally forwards all requests to a specified upstream server. This is in contrast to reverse proxy mode, in which mitmproxy forwards ordinary HTTP requests to an upstream server.

command-line -F http[s]://hostname[:port]
mitmproxy shortcut F
